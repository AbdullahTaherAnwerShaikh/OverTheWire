# OverTheWire Writeups

This repository contains my notes and writeups for the OverTheWire wargames as I work through them. The goal is to document the techniques, Linux commands, and security concepts learned during each challenge while building a personal knowledge base.

## Repository Structure

```
OverTheWire/
├── Bandit/
├── Natas/
├── Leviathan/
├── Krypton/
├── ...
```

Each game has its own directory containing level-by-level writeups.

## Writeup Format

Each level includes:

* Objective
* Commands used
* Key concepts learned
* Important notes (when applicable)

Passwords are intentionally omitted.

## Purpose

This repository serves as:

* A record of my progress through the OverTheWire wargames.
* A reference for Linux and cybersecurity techniques.
* Documentation of my hands-on practice while developing penetration testing and red teaming skills.

## Topics Covered

* Linux command line
* File permissions
* SSH
* Text processing
* Encoding and decoding
* Compression and archives
* Networking
* Processes and services
* Scripting
* Git
* Basic exploitation concepts

## Disclaimer

These writeups are intended for educational purposes only. All challenges are completed within the official OverTheWire training environment.
